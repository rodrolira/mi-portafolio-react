# Mi Portafolio - Desarrollador Full Stack

Portafolio profesional tipo CV interactivo desarrollado con React + Vite.

## 🚀 Tecnologías utilizadas

- React 19
- Vite 6
- CSS Modules
- JavaScript ES6+

## ✨ Características

- Diseño responsive y moderno
- Animaciones al hacer scroll
- Barras de progreso interactivas
- Componentes reutilizables
- Estructura profesional de carpetas

## 📦 Instalación

```bash
# Clonar el repositorio
git clone https://github.com/rodrolira/mi-portafolio-react.git

# Entrar al proyecto
cd mi-portafolio-react

# Instalar dependencias
npm install

# Iniciar en modo desarrollo
npm run dev
